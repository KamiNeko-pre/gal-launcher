# 行为准则 / Code of Conduct

## 我们的承诺

为了营造一个开放和友好的社区环境，我们承诺尊重所有参与者——无论其经验水平、性别、性取向、残疾、外表、体型、种族、年龄、宗教信仰或国籍。

本项目服务于中文 Galgame 社区。我们理解并且尊重视觉小说文化圈的多样性。请保持友善和建设性。

## 我们的准则

**鼓励的行为：**

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 建设性地接受批评
- 关注对社区最有利的事情
- 对其他社区成员表达同理心

**不可接受的行为：**

- 使用性暗示语言或图片、不受欢迎的性关注
- 挑衅、侮辱/贬损性评论、人身攻击
- 公开或私下骚扰
- 未经允许发布他人的私人信息
- 上传游戏本体、破解工具、或任何侵权/违法内容
- 在 Issue 中直接上传受版权保护的封面图或游戏素材（截图除外）

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受的行为采取适当和公平的纠正措施。

## 适用范围

本行为准则适用于所有项目空间（GitHub Issues、PR、Discussions 等）以及个人代表项目或社区时的公共空间。

## 举报

如遇到骚扰或其他不可接受的行为，请通过 GitHub Issues 联系项目维护者。所有投诉将被审查和调查，并产生被认为必要且适合情况的回应。维护者有义务对举报者信息保密。

## 署名

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
