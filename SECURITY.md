# Security Policy

## Supported Versions

This is an early personal project. Security fixes target the latest public version only.

## Reporting A Vulnerability

Please open a private security advisory on GitHub if available, or contact the maintainer privately before posting a public exploit.

## Scope

Relevant issues include:

- Arbitrary code execution through imported backups or metadata
- Unsafe file path handling
- Unexpected network behavior
- Leaking local file paths or user library data

Out of scope:

- Third-party website availability
- Accuracy of third-party metadata
- Copyright disputes about user-selected external artwork
