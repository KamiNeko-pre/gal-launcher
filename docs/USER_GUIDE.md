# Gal Launcher 使用教程

这是一份给普通用户看的教程。你不需要会编程。

## 1. 下载

进入 GitHub 的 **Releases** 页面，下载最新版里的：

```text
Gal Launcher.exe
```

下载后双击打开。

如果 Windows 弹出安全提示，确认文件来自本项目 Release 后，选择继续运行。

## 2. 添加游戏

打开软件后，点击左下角的 `+` 按钮。

然后选择游戏目录里的启动文件。常见启动文件可能叫：

```text
game.exe
start.exe
launcher.exe
SiglusEngine.exe
```

也可能是：

```text
*.bat
*.cmd
*.lnk
```

选中后，游戏会加入你的本地游戏库。

## 3. 自动搜索资料

添加游戏后，软件会尝试搜索：

- 中文名 / 原名
- 会社
- 简介
- 发售日期
- 竖版封面
- 横版背景图
- Bangumi 评分

如果自动识别错了，不用慌。Galgame 的文件名经常和作品名不一致，这是正常情况。

## 4. 手动修正资料

进入游戏详情页，点击资料相关按钮。

你可以尝试输入：

- 日文原名
- 中文名
- 英文名
- 会社名

一般日文原名的识别率最高。

## 5. 修改封面

如果封面不对，可以：

- 使用“找横版图”搜索候选图
- 从候选图里手动选择
- 在编辑资料里选择本地图片

建议：

- 竖版封面用于游戏库封面墙
- 横版图用于启动页大背景

## 6. 启动游戏

点击游戏封面进入大图页面，然后点击“启动”。

软件会自动记录：

- 最近游玩时间
- 启动次数
- 总游玩时长

## 7. 收藏墙 / 游戏库

左侧栏的游戏库按钮可以进入封面墙。

这里会像 Steam 一样显示所有游戏的竖版封面。点击任意封面，会回到该游戏的横版启动页。

## 8. 备份和恢复

如果你要换电脑，建议先导出备份。

备份里可能包含你的本地游戏路径，所以不要随便发给别人。

恢复备份后，如果游戏路径变了，可能需要重新编辑启动文件路径。

## 9. 数据保存在哪里

默认保存在：

```text
%APPDATA%\gal-launcher\library
```

里面包括：

- 游戏列表
- 本地路径
- 封面缓存
- 背景图缓存
- 游玩记录

这些都是你的个人数据。

## 10. 常见问题

### 添加游戏后识别错了怎么办？

手动搜索日文原名，或者打开编辑资料自己修改。

### 搜不到图怎么办？

可以手动选择本地图片。部分新作、冷门作品可能暂时没有合适的在线图片。

### 游戏启动了但时长没记录怎么办？

有些游戏启动方式比较特殊，比如先打开启动器，再打开真正的游戏。可以尝试选择真正运行游戏的 exe，而不是补丁启动器或设置工具。

### 可以管理非 Galgame 吗？

可以。只要是本地可执行文件，理论上都能添加。但资料搜索更偏向视觉小说和 Galgame。
